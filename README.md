# Práctica Pygame 2D

Proyecto de desarrollo de videojuego 2D orientado a la práctica y consolidación
de habilidades en Python y Pygame.

## 🛠️ Tecnologías utilizadas

- **Python 3.13.7**
- **Pygame 2.6.1**

## 🎯 Objetivos de aprendizaje

- Consolidar conocimientos de Python orientado a objetos
- Explorar las capacidades de Pygame para el desarrollo de videojuegos 2D
- Implementar mecánicas de juego como animaciones, movimiento y colisiones
- Aplicar buenas prácticas de organización y estructura de proyectos

## ▶️ Ejecución

```bash
python main.py
```
